## üöÄ Getting Started

1. Clone the repository
```bash
git clone https://github.com/abhay-byte/mad.git
```

2. Navigate to any experiment
```bash
cd experiment_X
```

3. Install dependencies
```bash
# For Flutter projects
flutter pub get

# For Android projects
./gradlew build
```

4. Run the application
```bash
# For Flutter projects
flutter run

# For Android projects
./gradlew installDebug
```
