### Components:
1. **Search Bar**
   - Floating search bar at top
   - Material3 outlined style
   - Search icon and clear button
   - Expandable for search results

2. **Map View**
   - Full-screen map component
   - Custom markers for different place types
   - Clustering for multiple markers
   - Smooth camera animations

3. **Current Location Button**
   - FAB style
   - Bottom-right position
   - Loading animation when locating
   - Color indication for accuracy

4. **Bottom Sheet**
   - Expandable bottom sheet
   - Place details with images
   - Action buttons (directions, share, save)
   - Smooth drag behavior
